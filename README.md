# MOroom reservation v8

- 日本語表示
- 装置名は MOVPE 系6種類に固定
- 新規予約・変更・削除にパスワード必須
- 使用目的は空欄でも保存可能
- 予約カードの時刻表示は `09:00-17:00` のみ。`（8時間）` のような括弧表示は出しません。
- 予約カードは文字を小さくし、`/`、`、`、`。`、`・` などの直後で改行します。
- 上部タブ末尾に `メンテ情報` を表示します。
- `メンテ情報` タブは表示専用です。フォームの装置欄には出ません。
- フォームのメンテ情報欄は、まず `なし` / `メンテ` / `エピ` を選択します。
- `メンテ` を選択した場合だけ、`原料交換` / `重故障` / `除害停止` / `定常メンテ` から選択できます。
- `エピ` を選択した場合は、詳細項目なしで `エピ` として登録されます。
- `メンテ情報` タブには、各装置でメンテ情報が登録された予約だけが、装置をまたいでまとめて表示されます。
- 旧データ互換：`除害停止メンテ` は `除害停止`、`重故障メンテ` と旧 `メンテ` は `重故障` として表示します。

## GitHub Pagesでの使い方

zipを解凍して、中身のファイルをリポジトリ直下へアップロードしてください。

上書き対象：

- `index.html`
- `config.js`
- `script_v5.js`
- `style_v5.css`
- `apps_script_backend.gs`
- `README.md`

`config.js` に実際の `API_URL` を設定済みの場合は、そのURLを消さずに残してください。

## Google Apps Scriptを使っている場合

`apps_script_backend.gs` をApps Script側に貼り替えて、Webアプリを再デプロイしてください。

既に予約データがある場合、`setup()` は実行しないでください。既存データが消えます。
