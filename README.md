# MOroom reservation maintenance v6

- 日本語表示
- 装置名は MOVPE 系6種類 + 「全体のメンテ」
- 新規予約・変更・削除にパスワード必須
- 使用目的は空欄でも保存可能
- メンテ情報チェック欄を追加
  - 除害停止メンテ
  - 重故障メンテ
  - エピ
- 予約カードの文字を小さくし、`、` `。` `/` `・` などの後で改行
- 予約カードでは「名前」「使用目的」などのラベルを出さず、内容だけを表示

## 更新方法

GitHub Pagesでは、このzipを解凍して中身のファイルをリポジトリ直下へアップロードしてください。

既にGoogle Apps Script / Google Sheetsで共有運用している場合、Apps Script側は `apps_script_backend.gs` に上書きしてから、Webアプリを再デプロイしてください。

既に予約データが入っている場合は、Apps Scriptの `setup()` を実行しないでください。`setup()` はシートを初期化するため、既存データが消えます。

`config.js` に実際の `API_URL` が入っている場合は、そのURLを消さずに残してください。
